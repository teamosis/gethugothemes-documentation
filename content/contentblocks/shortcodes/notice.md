## Notice Shortcode

```md
{{</* notice note */>}}This is a simple note{{</*/ notice */>}}
```

You can use this shortcode to display a notice. The notice can be styled with the following tags: `note`, `warning`, `tip`, `info`. You can use markdown in the notice.
